***
### setup

* .env : password , host , dbname set

```
POSTGRES_DATABASE="__DBNAME__"
POSTGRES_USER="__USER__"
POSTGRES_PASSWORD="__PASSWORD__"
POSTGRES_HOST="__DBHOST__"
POSTGRES_PORT=5432
```

***
### Start

```
# dev-start
yarn dev

# start
yarn build

yarn start

```
